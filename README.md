# Alex Moore's Blog

This is the repository for Alex Moore's personal blog, focusing on cloud and IT infrastructure topics.

## Getting Started

To run the blog locally, you need to have Hugo installed.

1.  Clone the repository:
    ```bash
    git clone https://github.com/alexjmoore/blog.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd blog
    ```
3.  Start the Hugo development server:
    ```bash
    hugo server -D
    ```
    The site will be available at `http://localhost:1313/`.
